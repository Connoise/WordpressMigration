Frog Budget

Frog Budget 🐸

Code Project

A modern cross-platform personal budget tracking app — built to actually be used, not abandoned after two weeks. Runs as a PWA on Android and in any browser.

Features

  • ✓Cross-platform (browser + installable Android PWA)
  • ✓Real-time sync across devices via Supabase
  • ✓Category budgets with income percentage allocation
  • ✓Purchase logging with name, amount, date, category
  • ✓Budget analysis and overspending alerts
  • ✓Visual analytics — spending trends and category breakdowns
  • ✓End-of-month spending projections
  • ✓CSV export
  • ✓Dark mode

Stack

React TypeScript Supabase Vite Tailwind PWA

Why

Most budget apps are either too complex, subscription-gated, or just annoying to use. Frog Budget is personal infrastructure — simple, owned, and works offline-first.

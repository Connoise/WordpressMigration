Chuuning-Byou

中二病 — "8th Grader Syndrome" (delusions of power and uniqueness)

Code Project — VST3 Plugin

A Windows VST3 vocal processing plugin combining real-time pitch correction with dynamic harmonic region shaping. Built for vocals that need character — supports both natural correction and intentionally robotic tuning.

Signal Flow

Mono Input → Pitch Detection → Target Selection (MIDI / Scale)
→ Pitch Correction → Harmonic Region Processing
→ Wet/Dry Mix → Output Limiter → Mono Output

Features

  • Low-latency pitch detection and correction with adjustable retune speed
  • Two pitch-shift algorithms: time-domain (natural) and granular (robotic)
  • MIDI input support — highest held note sets target pitch
  • 5 dynamic harmonic regions, centered at multiples of detected f₀
  • Per-region gain and Q (bandwidth) controls
  • Automatic fallback to scale-based correction when no MIDI held
  • Harmonic processing bypasses outside 80Hz–12kHz vocal range

Stack

C++ JUCE VST3 Windows DSP

Purpose

Built to serve Agoe the Idol's live and studio vocal chain. The harmonic shaping approach came from wanting pitch correction that feels expressive and sculptural rather than transparent and corrective.
